# 원티드 프리온보딩 프론트엔드 - 사전 과제

## 프로젝트의 실행방법

```bash
#레포지토리 clone
git clone https://github.com/meljaraks/wanted-pre-onboarding-frontend.git

# 라이브러리 설치
npm install

# 클라이언트 시작
npm start
```

## 사용한 기술스택

- React
- tailwindCSS
- axios
- react-router-dom
